rfDynHUD

rfDynHUD means rFactor Dynamic HUD and is a feature rich, configurable and extendable dynamic HUD system for rFactor (1 and 2) and now also other sims.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the rfDynHUD page

rfDynHUD is configured through ini files, which are created and modified using the provided editor, which is kinda self-documentary. Please read the readme with care to learn how to use this plugin.

The internal code structure has been tweaked to allow for better binding capabilities to other sims.

Your personal live overlays

Create your own custom overlays of live data for rFactor with as much or as little information as you want. rfDynHUD comes with its own intuitive editor which allows you to define when and which overlays are shown to you. These overlays can be heavily customized to suit your needs. You can create different overlays for each session, car and mod.

Many widgets, not just for drivers

Several overlays sets and dozents of single widgets are included with rfDynHUD. Ecclestone TV simulates TV-coverage overlays, while other sets are better for drivers or for race directors.

Build your own HUD

rfDynHUD is Open Source and licensed under GNU General Public License. The package includes an SDK and a rich tutorial showing you step by step, how to build your very own Widgets in Java. To join the development have a look at the source code and join us..
